Cursor Sample
     General Information

Syntax: codeID resType dataFile
Where codeID is an integer or string.

The LoadResPicture function uses 3 constants (keywords):
  • vbResBitmap = 0
  • vbResIcon = 1
  • vbResCursor = 2
All other graphic formats must use a user-defined resType


The Resource Compiler (rc.exe) must have direct access to the dataFile either by having the complete path, or having the resource in the same folder. When using the complete path, there must be two slashes "\\" used after the drive letter: "c:\\vb\resources.filename.xxx".


Format for the precompile resource file (filename.RC) entry using an Identifier:

Hand CURSOR "hand.cur"


Usage in the program...

'Don't forget the Custom Property
Picture1.MousePointer = 99
Picture1.MouseIcon = LoadResPicture("Hand", vbResCursor)



Format for the precompile resource file (filename.rc) entry using an Index Number and substituting the constant's value: vbResCursor = 0

101 CURSOR "hand.cur"


In the program...

Picture1.Picture = LoadResPicture(101, 2)
